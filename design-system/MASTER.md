# جاسر (Jasir) — نظام التصميم الرئيسي (MASTER)

> مصدر الحقيقة الوحيد لتصميم واجهة جاسر. مشتقّ مباشرةً من **هوية_جاسر_البصرية.html** (المرجع الملزم).
> القاعدة: أي ملف في `pages/` يتجاوز هذا الملف لصفحته فقط؛ إن لم يوجد ملف صفحة → يُطبّق MASTER حصريًا.
> المنصّة: **Flutter (Material 3)** · اتجاه **RTL عربي** · iOS أولًا (TestFlight) · مودان فاتح/داكن.
> النطاق الحالي: **الشاشة الرئيسية (Home) — services_grid_screen.dart**.

---

## التصنيف (Category)
مساعد شخصي/عائلي + صحة + عبادة. القيم المحرّكة للتصميم: **ثقة · طمأنينة · عناية · وقار**.
الطموح: "هوية عالمية" — انضباط بصري بمستوى تطبيقات الدرجة الأولى، لا "قوس قزح" افتراضي.

## PATTERN — بنية الشاشة الرئيسية
**النمط:** Greeting + Priority ("اليوم") + Bento مجمّع دلاليًا (يتفوّق على شبكة عمودين موحّدة).

ترتيب الأقسام (من أعلى لأسفل):
1. **منطقة الترحيب (Greeting Zone):** شعار جاسر مصغّر + تحية بخط **Amiri** «صباح الخير، علي» + التاريخ الهجري + سطر ملخّص اليوم (Tajawal ثانوي). خلفية هادئة (Surface) لا هيدر Teal ثقيل — الهدوء أولًا.
2. **منطقة الأولوية «اليوم» (Priority/Hero):** بطاقة بارزة واحدة تحمل **رسالة الصباح** (قلب جاسر) + شريط أفقي لعناصر اليوم القادمة (الموعد التالي · دواء مستحق). لمسة ذهبية (tertiary) للإبراز. هذه المنطقة تجيب «ماذا يهمّني الآن؟» قبل أي شبكة خدمات.
3. **الخدمات المجمّعة دلاليًا (Bento Groups):** أربع مجموعات معنونة، كل مجموعة عائلة لون واحدة (انظر استراتيجية الألوان). بلاطات ضمن كل مجموعة (لا شبكة 14 بلاطة موحّدة).
4. **مجموعة الروحانيات:** معالجة متميّزة (Teal عميق + Amiri) في أسفل الشاشة كخاتمة وجدانية.

> رسالة الصباح تُرسي منطقة الأولوية كـ Hero. «أذكار المؤمن» تسكن مجموعة الروحانيات.

## STYLE — الاتجاه البصري
**الاسم:** Calm Bento — Organic/Biophilic Minimalism (بينتو هادئ بروح عضوية).
**الكلمات المفتاحية:** مساحات بيضاء سخيّة · زوايا ناعمة · ظلال منخفضة مصبوغة بالـTeal · تجميع دلالي · سكينة.
**الأفضل لـ:** تطبيقات العناية/الصحة/العائلة التي تحتاج ثقة وهدوء لا إثارة — يمنح إحساس «مساحة تتنفّس» ويقتل فوضى الألوان الـ14 الحالية.

## استراتيجية الأكسنت — حلّ قوس قزح الـ14 لونًا
**القاعدة الحديدية:** لا لون عشوائي لكل بلاطة. **4 مجموعات دلالية، كل مجموعة عائلة Hex واحدة مشتقّة من الهوية.**
اللون يُطبَّق على **حاوية المجموعة + خلفية الأيقونة فقط**؛ عناوين البلاطات كلها `onSurface`. الأيقونات SVG أحادية اللون داخل خلفية دائرية/مربّعة ناعمة بلون المجموعة.

| المجموعة | الخدمات | عائلة اللون | حاوية البلاطة (Light) | لون الأيقونة |
|---|---|---|---|---|
| **صحة** | الأدوية · القياسات · المواعيد | Teal أساسي | `#CDEDE3` (primaryContainer) | `#0E7C6C` |
| **عائلة** | العائلة · العمال · المقاسات | ذهبي دافئ | `#FBF3E0` (tertiaryContainer) | `#B4820F` |
| **شؤون يومية** | السيارات · الوثائق · الديون · التسوق · الجدول · الروابط | محايد مصبوغ Teal | `#EEF3F0` (surfaceContainer) | `#3E5149` |
| **روحانيات** | رسالة الصباح · أذكار المؤمن | Teal عميق + ذهبي | `#0B5E57` (خلفية داكنة) نص `#F7F0DC` | ذهبي `#FBBF24` |

**لماذا:** المجموعات تعطي هيكلًا معرفيًا (المستخدم يمسح بالمعنى لا باللون)، تحترم توزيع 60/30/10، وتجعل الشاشة تبدو مصمّمة لا مجمّعة. مجموعة الشؤون اليومية محايدة عمدًا كي **تنحسر** فتبرز الصحة والروحانيات.

**الداكن (حاويات المجموعات):** صحة `#12312A` · عائلة `#3A2E12` · شؤون يومية `#1E2833` · روحانيات `#0B5E57`.

## TYPOGRAPHY — عبر google_fonts (^6.2.1 مثبّت)
**الاقتران:** **Tajawal** (كل الواجهة والأرقام) + **Amiri** (اللحظات الوجدانية فقط).
**المزاج:** Tajawal عصري هندسي واضح للقراءة السريعة والأرقام؛ Amiri أصيل تقليدي يمنح دفئًا وجدانيًا. كلاهما دعم عربي ممتاز وضبط RTL.
**Amiri محجوز لـ:** اسم «جاسر» · تحية الصباح («صباح الخير، علي») · الأدعية · عناوين/متون أذكار المؤمن ورسالة الصباح. **لا تستخدم Amiri للأرقام أو أزرار الواجهة.**

الاستيراد (Flutter):
```dart
// pubspec: google_fonts: ^6.2.1
import 'package:google_fonts/google_fonts.dart';
final tajawal = GoogleFonts.tajawalTextTheme();
final amiri   = GoogleFonts.amiri();
```

## KEY EFFECTS — لغة الظل/النصف قطر + قيم الحركة

### النصف قطر (Radius)
- حاوية مجموعة Bento: **24**
- بلاطة خدمة / بطاقة أولوية: **18**
- شارة/شريحة داخلية · خلفية أيقونة: **12**
- زر: **14** · Bottom sheet: **28** · حاوية دائرية (avatar/FAB): **999**

### الظل (منخفض، مصبوغ Teal — لا ظلال سوداء قاسية)
- **Level 1 (بلاطات):** `offset(0,2) blur 10 spread -3` بلون `rgba(11,94,87,0.06)` + `rgba(20,32,28,0.04)`
- **Level 2 (بطاقة الأولوية/المرفوع):** `offset(0,8) blur 24 spread -6` بلون `rgba(11,94,87,0.10)`
- **الداكن:** لا ظل — استخدم فرق لون السطح (surfaceContainerHigh) للارتفاع.

### الحركة (Flutter Durations + Curves — لـ motion-craft)
| التفاعل | المدة | المنحنى |
|---|---|---|
| Micro (ripple · تبديل أيقونة) | **120ms** | `Curves.easeOut` |
| ضغط البلاطة (scale → 0.97) | **180ms** | `Curves.easeOutCubic` |
| ظهور/كشف (stagger البلاطات · sheet) | **280ms** | `Cubic(0.05, 0.7, 0.1, 1.0)` (emphasized decelerate) |
| تأخير التتابع بين البلاطات (stagger) | **40ms** | — |
| انتقال الصفحة (RTL: انزلاق من اليسار + تلاشٍ) | **300ms** | `Curves.easeOutCubic` |
| **سبينر جاسر** (أشعة الشمس تتلاشى بالتتابع) | دورة **1.2s** خطّي، 12 شعاع، delay ‎-0.1s/شعاع | لودر الهوية الرسمي |

**تفاعل الضغط:** بلاطات الخدمات تنكمش scale 0.97 + رفع ظل خفيف عند اللمس (لا hover في iOS). ريبل بلون `primary` بشفافية 8%.

## AVOID — مضادّات النمط + الحظر الكوني

### خاص بالفئة (صحة/عائلة/عبادة)
- **لا قوس قزح:** ممنوع لون عشوائي لكل بلاطة (المشكلة الحالية) — الألوان من المجموعات الأربع فقط.
- ممنوع الأزرق `#5B8DEF` / البنفسجي `#7B61D1` / السماوي / البني `#8B6C61` الحالية — خارج الهوية كليًا.
- لا تدرّجات AI بنفسجية/وردية، ولا نيون أو ألوان صاخبة — تكسر الطمأنينة.
- لا حركة عدوانية/مرتدّة (bounce/overshoot) — الحركة هادئة انسيابية فقط.
- ممنوع تدوير الشعار أو تغيير ألوانه أو وضعه على خلفية مزدحمة (نصّ الهوية).
- الذهبي لمسة نادرة (10%) — لا خلفيات ذهبية كبيرة.
- سياق العبادة (أذكار/رسالة الصباح): وقار وبساطة — لا زخرفة مبالغة ولا حركة لعوب.

### الحظر الكوني
- **لا إيموجي كأيقونات** — SVG فقط (Heroicons/Lucide، أو أيقونات مخصّصة أحادية اللون بلون المجموعة).
- تباين نص أقل من **4.5:1** ممنوع (تحقّق من الذهبي فوق الأبيض ومن النص الثانوي).
- لا حالات تركيز مفقودة — كل عنصر تفاعلي له focus/pressed مرئي.
- لا خطوط افتراضية (Material الافتراضي) — طبّق Tajawal/Amiri عبر google_fonts على مستوى الـTheme.

```

[ ] الألوان من ColorScheme/ThemeExtension فقط — صفر Hex مكتوب داخل الودجت
[ ] الخدمات ملوّنة بالمجموعات الأربع — لا لون عشوائي لكل بلاطة
[ ] Tajawal + Amiri مطبّقان عبر google_fonts على TextTheme
[ ] Amiri محصور في التحية/الأدعية/الأذكار فقط
[ ] تباين النص ≥ 4.5:1 (فاتح وداكن) — تحقّق من الذهبي والنص الثانوي
[ ] أهداف اللمس ≥ 48dp (البلاطات والأزرار والأيقونات القابلة للنقر)
[ ] cursor/ريبل + حالة ضغط (0.97) مرئية على كل قابل للنقر
[ ] انتقالات الحركة 120–300ms بالمنحنيات المحدّدة أعلاه
[ ] RTL صحيح: EdgeInsetsDirectional · start/end لا left/right · انزلاق الصفحة من اليسار
[ ] Dynamic Type: النصوص تحترم MediaQuery.textScaler دون قصّ أو تراكب
[ ] احترام تقليل الحركة: MediaQuery.disableAnimationsOf(context) → مدد الحركة = صفر
[ ] المودان الفاتح/الداكن يعملان على كل عنصر (لا Hex ثابت يكسر الداكن)
[ ] استجابة العرض: 375 / 768 / 1024 / 1440 (iPhone SE → iPad)
[ ] سبينر جاسر (أشعة الشمس) هو اللودر الوحيد — لا CircularProgressIndicator افتراضي
```
